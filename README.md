# Tetris (Raylib, Windows)

Hello, this game is made especially for Windows using raylib. It has external dependencies, so it may not work by simply copying and running the code. The easiest way to run it is to open the Visual Studio solution and build from there.

The main screen and the game screen use custom backgrounds designed to align closely with the UI, so the code and visuals feel in sync.

HOPE YOU LIKE IT.

## Key Features

- Single-player Tetris gameplay with level progression (up to level 8).
- Next-piece preview for better planning.
- Pause/resume with an on-screen button.
- Simple controls: move, rotate, and fast-fall within seconds.
- Eye-catching visuals with a consistent theme and custom backgrounds.
- Sound effects for start, pause, line clear, and game over.

## Controls

- Left / Right Arrow: Move piece
- Up Arrow: Rotate piece
- Down Arrow: Fast fall
- Mouse click on pause circle: Pause/Resume

## Build & Run (Windows)

1. Open `Tetris.sln` in Visual Studio.
2. Restore NuGet packages if prompted.
3. Build and run the project.

## Other Info

- Programming language: C++
- Game engine: Raylib
- Platform: Windows (Visual Studio project included)
- Art assets: Custom backgrounds created in external tools
- Audio: Included WAV sound effects
